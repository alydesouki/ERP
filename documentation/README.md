## Quick Start

```bash
# Install dependencies
pnpm install

# Run API server (port 5001)
pnpm --filter @workspace/api-server run dev

# Run POS frontend (port 5000)
pnpm --filter @workspace/pos run dev

# Full typecheck
pnpm run typecheck

# Push DB schema (dev only)
pnpm --filter @workspace/db run push
```
